WireGuard peer management page

This page is used to add, delete and download WireGuard peer configuration files.

The "LAN-traffic-only" config file only sends traffic for 192.168.2.0-192.168.2.255 addresses over the WireGuard tunnel, and the "All traffic" config file sends all traffic over the WireGuard tunnel.
